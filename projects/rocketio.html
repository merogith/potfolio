RocketIO

Command one of 14 world powers in a real-time hex-grid war — no engine, everything procedural.

Play the game GitHub
14factions / 70 units 5victory modes proceduralmaps & audio

What it is

A real-time strategy game on a hex grid. Expand your borders, run a missile economy (factories build, launchers fire, anti-air intercepts), and outmaneuver up to six doctrine-driven AIs that adapt to your strategy — all under fog of war. Five victory modes, plus a single-player campaign and tutorial.

The interesting part

There's no engine, no framework, and no asset pipeline. Built with vanilla JS ES modules and the HTML5 Canvas, it generates everything at runtime — terrain (gradient-noise continents, archipelagos, inland seas), units, projectiles, and even the audio, via WebAudio with an adaptive score. Vite handles bundling, Vitest plus GitHub Actions handle testing/CI, and a separate balance-simulator tool tunes the factions.
