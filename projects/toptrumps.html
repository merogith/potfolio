What it is

A browser Top Trumps card game: local two-player pass-and-play or single-player against Easy / Medium / Hard bots, across six themed decks. The hard bot tracks every card already played to choose the statistically optimal stat. Pass-and-play uses privacy gates between turns, and the whole thing is mobile-first and accessible.

The interesting part

The engine takes an immutable GameSnapshot and returns the next one — no DOM, no globals — which is exactly why it's exhaustively testable. Unit tests cover the state machine, bot logic, and seeded RNG; smoke tests play a full match in a DOM simulator; and Playwright drives a complete game end-to-end in a real browser. It ships as ~20 KB gzipped with zero runtime dependencies, deployed automatically via GitHub Actions.
